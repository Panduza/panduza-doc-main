# How to add a korad ka3005 in tree.json

Attribute ***name*** can be choosen

```json
{
    "ref": "korad.ka3005",
    "name": "Korad KA3005",
    "settings": {
        "serial_baudrate": 9600,
        "usb_vendor": "1046",
        "usb_model": "20497"
    }
}
```
