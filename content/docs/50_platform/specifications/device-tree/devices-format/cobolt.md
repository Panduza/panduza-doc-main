# How to add a cobolt S0501 in tree.json

Attribute ***name*** can be choosen

```json
{
    "ref": "cobolt.s0501",
    "name": "Cobolt S0501",
    "settings": {
        "serial_baudrate": 9600,
        "usb_vendor": "9692",
        "usb_model": "6"
    }
}
```
