# How to add a voxpower inhibitor in tree.json

Attribute ***name*** can be choosen

```json
{
    "ref": "panduza.voxpower_inhibiter",
    "name": "my_voxpower_inhibiter",
    "settings": {
        "serial_baudrate": 9600,
        "usb_vendor": "9025",
        "usb_model": "0067"
    }
}
```
